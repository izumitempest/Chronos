# Mandate

Location-verified attendance for Nigerian universities — a hackathon prototype for Wema Bank's Hackoholics 7.0, piloted at Godfrey Okoye University.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Demo flow

1. Pick **Student** → tap **Check in** → watch the calm location confirmation
2. Switch to **Lecturer** → see students quietly appear on today's roster
3. Switch to **Admin** → view department compliance chart, click a course bar for student detail

Use the **Demo** pill in the bottom-right corner to toggle check-in outcomes and roster speed.

## Stack

React, TypeScript, Tailwind CSS, Vite. In-memory mock store — no backend.
